Interpolation is a method of constructing new data points within the range of a discrete set of known data points. Image interpolation refers to theâ€œguessâ€of intensity values at missing locations.

**å•çº¿æ€§æ’å€¼**

![](\Images/38-01.png)

$${y-y_1\over x-x_1}={y_2-y_1\over x_2-x_1}$$

$$y={x_2-x\over x_2-x_1}y_1+{x-x_1\over x_2-x_1}y_2$$

åˆ†å­å¯ä»¥çœ‹ä½œ $x$ ä¸ $x_1$, $x_2$ è·ç¦»çš„æƒé‡ï¼Œ$y_1$, $y_2$ æ˜¯åŠ æƒç³»æ•°ã€‚

**åŒçº¿æ€§æ’å€¼**

def bilinear_kernel(in_channels, out_channels, kernel_size):
    #åˆ©ç”¨åŒçº¿æ€§æ’å€¼åˆå§‹åŒ–è½¬ç½®å·ç§¯æ ¸
    factor = (kernel_size + 1) // 2
    if kernel_size % 2 == 1:
        center = factor - 1
    else:
        center = factor - 0.5
    og = (torch.arange(kernel_size).reshape(-1, 1),
          torch.arange(kernel_size).reshape(1, -1))
    filt = (1 - torch.abs(og[0] - center) / factor) * \
           (1 - torch.abs(og[1] - center) / factor)
    #å¯ä»¥ç®€å•ç†è§£ä¸ºog[0]/[1] - centeræ˜¯æ’å€¼å‡ºçš„ç‚¹ä¸ä¸­å¿ƒçš„è·ç¦»
    #/ factoræ˜¯æ•£ç‚¹è·ç¦»ä¹‹äºæ€»è·ç¦»çš„æ¯”ä¾‹
    #1 - æ˜¯æŠŠæ¯”ä¾‹å€’ç½®ï¼Œä½¿ä¸­å¿ƒæƒé‡é«˜ï¼Œä¸¤è¾¹æƒé‡ä½
    #ä¹Ÿå°±æ˜¯è¯´æ’å€¼è¶Šé ä¸­å¿ƒçš„ç‚¹è¶Šæ¥è¿‘è¢«å·ç§¯ä¹‹å‰åƒç´ çš„å€¼
    weight = torch.zeros((in_channels, out_channels,
                          kernel_size, kernel_size))
    weight[range(in_channels), range(out_channels), :, :] = filt
    return weight
